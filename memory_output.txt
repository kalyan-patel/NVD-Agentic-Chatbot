:  ### CVE-2023-0620

**Date**: March 30, 2023

**Vulnerability Description**:
HashiCorp Vault and Vault Enterprise versions 0.8.0 through 1.13.1 are susceptible to an SQL injection attack. This vulnerability occurs when configuring the Microsoft SQL (MSSQL) Database Storage Backend. Specifically, certain parameters are not sanitized correctly when passed to the user-defined MSSQL database during the configuration of the MSSQL plugin.

**Impact**:
An attacker could exploit this vulnerability by modifying the unsanitized parameters to execute malicious SQL commands against the affected database. This could lead to unauthorized access or manipulation of data stored in the MSSQL database.

**Resolution**:
The issue has been addressed in the following versions:
- 1.13.1
- 1.12.5
- 1.11.9

Users are advised to upgrade to one of these fixed versions to mitigate the risk associated with this vulnerability.
